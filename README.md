## Dependencies

- Yarn
- Grunt
- SASS

## Running the project

To run this small project pull the repository using git:

<code>
    git clone {repo}
</code>

Then run yarn install:

<code>
    yarn install
</code>

if you want to run any changes you can then run the watcher to compile SCSS:

<code>
    grunt watch
</code>
